To install it:

1.  Download that file and unzip it to a place of your choosing.

2.  In Matlab, go to File > Set Path... 

3.  Remove any paths for older version of Groundswell.

3.  Still in File > Set Path, click on "Add with Subfolders..." and select the groundswell_rel_<whatever> folder that was created when you unzipped the .zip file.

4.  Restart Matlab.

5.  Type "groundswell" or "roving" at the Matlab prompt to run the programs.
